SBOMScope Documentation

SBOMScope is a client-side SBOM visualizer for SPDX and CycloneDX inputs. You can load files directly, paste a URL, or use the built-in sample files.

How to use SBOMScope

  • Upload an SBOM file using the upload box.
  • Drag and drop files directly into the upload area.
  • Paste a remote SBOM URL and click Load.
  • Switch between Summary, Components, Graph, Licenses, Vulnerabilities, and Raw Source views.

Supported formats

  • CycloneDX JSON and XML
  • SPDX JSON, YAML, Tag-Value
  • SPDX 3.0 JSON-LD

For more details, see the repository README.
